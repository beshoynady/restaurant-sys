# نظرة عامة على النظام المحاسبي

يوفر هذا الموديول:

- محرك محاسبي متكامل للمطاعم.
- تتبع المرتبات، المبيعات، المشتريات، المصروفات، الأصول، والنقدية.
- إنشاء قيود اليومية تلقائيًا بناءً على الأنشطة.
- دعم متعدد الفروع والعملات.
- سجل تدقيق والتوافق المالي.

## خطوات سريعة

1. تعريف الحسابات في نموذج `Account`.
2. إعداد `AccountingSettings` لتحديد حسابات التحكم وخريطة النشاط.
3. تعريف `AccountingPeriod`.
4. إنشاء المعاملات عبر أنشطة النظام (مبيعات، رواتب، إلخ).
5. إنشاء قيود اليومية تلقائيًا ويمكن تثبيتها.
6. استخراج التقارير والقوائم من القيود المثبتة.

## ملاحظات

- دائمًا عيّن `createdBy` لأغراض التدقيق.
- استخدم `CostCenter` لتقارير تحليلية مقسمة إذا كان مفعلًا.
- لا تحدد الحسابات بشكل صلب في الموديولات التشغيلية.
